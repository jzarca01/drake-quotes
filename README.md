# drake-quotes

Because surprisingly I couldn't find any on the Internet

## Installation

```shell
npm install drake-quotes
```

## Usage

```javascript
const { getQuote } = require('drake-quotes');
```

## Methods

```javascript
getQuote();
getQuote(category);
```
